## Вопросы бота (итоговый список)

1. Ты дочитал результат до конца? *(кнопки: Да / Частично / Нет, бросил)*
2. Что ты почувствовал во время чтения? *(кнопки: Узнавание "это точно я" / Удивление / Скорее скепсис / Смешанное)*
3. Насколько точно описание тебя? *(кнопки: 1–5)*
4. Какой раздел описал тебя точнее всего? *(кнопки: Характер / Любовь / Деньги и карьера / Здоровье / Периоды жизни)*
5. Зачем ты пришёл к этому сервису? *(кнопки: Просто интересно / Хотел лучше понять себя / Есть конкретный вопрос / Посоветовали)*
6. Что не зашло или показалось общим? *(текст)*
7. Ты доверяешь результату? *(кнопки: Да, верю / Скорее да / Скорее нет / Нет)*
8. Важно ли тебе понимать, на основе чего строится анализ? *(кнопки: Да, хочу понимать метод / Нет, мне важен результат / Не думал об этом)*
9. Ты бы отправил результат другу? *(кнопки: Да / Зависит от результата / Нет)*
10. Сколько готов заплатить за такой анализ? *(кнопки: Только бесплатно / до 300р / 300–1000р / больше 1000р)*
11. Пользовался раньше чем-то похожим? *(кнопки: MBTI/Gallup / Астрология / Нумерология / Ничего похожего / Несколько из этого)*
12. Есть ещё мысли или пожелания? *(текст, можно пропустить)*

## Как запустить бота

1. Создай бота через @BotFather командой `/newbot`, получи токен
2. Узнай свой Telegram chat_id через @userinfobot
3. Добавь в `.env`:
   ```
   TELEGRAM_BOT_TOKEN=токен_от_BotFather
   TELEGRAM_ADMIN_CHAT_ID=твой_chat_id
   APP_URL=https://твой-домен
   ```
4. Запусти: `npm run bot`

Фидбек сохраняется в папку `feedback/` в формате `.md` и одновременно приходит тебе в Telegram в реальном времени.
